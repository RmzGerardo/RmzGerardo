### ⚡ Sobre Mí

Soy un desarrollador de software con una fuerte inclinación por la lógica de sistemas, la gestión de bases de datos y la creación de arquitecturas sólidas. Me especializo en modernizar sistemas heredados y optimizar flujos de trabajo en el backend, priorizando siempre la calidad del código, el rendimiento y la eficiencia. Cuando no estoy estructurando bases de datos o configurando contenedores, disfruto explorando el diseño web con estética *pixel-art* y analizando la genialidad rítmica y cultural detrás de temas como *Billie Jean* o *Beat It*.
